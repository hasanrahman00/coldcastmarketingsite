# Coldcast — Brand Kit

Reusable logo, wordmark, and banner assets. SVGs are the source of truth; PNGs
are rendered from them via `scripts/gen-brand-kit.mjs` (`node scripts/gen-brand-kit.mjs`).

## Logo (`/logo`)
| File | Use |
|---|---|
| `coldcast-logo.svg` | Primary mark — rounded blue tile + white swirl, transparent corners. Scales to any size. |
| `coldcast-logo-256/512/1024.png` | Raster mark on transparent background (app icons, avatars, decks). |
| `coldcast-logo-square.svg` | Full-bleed square (no transparent corners) — for platforms that add their own rounding. |
| `coldcast-logo-300.png` | **300×300 — LinkedIn / social profile picture** (their recommended size). |
| `coldcast-logo-mono-white.svg` / `-512.png` | Swirl only, white, transparent — for dark, colored, or photo backgrounds. |

## Wordmark (`/wordmark`)
| File | Use |
|---|---|
| `coldcast-wordmark.svg` / `-1024w.png` | Logo + "Coldcast" lockup, **dark text** — light backgrounds. |
| `coldcast-wordmark-white.svg` / `-white-1024w.png` | Same lockup, **white text** — dark backgrounds. |

## Banners (`/banner`)
| File | Size | Use |
|---|---|---|
| `coldcast-linkedin-banner.png` | 1128×191 | LinkedIn Page cover. |
| `coldcast-linkedin-banner@2x.png` | 2256×382 | Retina/crisper LinkedIn cover (same ratio). |
| `coldcast-og-image.png` | 1200×630 | Social share / Open Graph preview (also copied to `/public/og-image.png`). |

## Favicons
The browser/search favicons live in `/public` (`favicon.ico`, `favicon-16/32/48/96.png`,
`favicon-192/512.png`, `apple-touch-icon.png`, `favicon.svg`) and are generated by
`scripts/gen-icons.mjs`.

| Brand light | `#4f7cf5` | Gradient start, accents |

| Brand dark | `#2546b8` | Deep blue |

Logo tile gradient: `linear-gradient(135deg, #4f7cf5 → #3257d6)`.
Brand/banner gradient: `linear-gradient(135deg, #233f9e → #3257d6 → #6d3df0)`.

## Fonts
- **Display / wordmark:** Space Grotesk (700). *(PNGs here are rendered with a system
  sans fallback since Space Grotesk isn't installed on the build machine — open the SVG
  with Space Grotesk available, or re-render after installing it, for an exact match.)*
- **Body:** Inter.

## Usage
- Keep clear space around the logo ≈ the height of the swirl's tile corner radius.
- Don't recolor the tile, stretch the mark, or add effects. On busy/dark backgrounds use
  the white mono mark or the white wordmark.
